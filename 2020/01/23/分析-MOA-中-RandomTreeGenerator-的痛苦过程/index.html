论文题目

我的论文题目是 “Analysis and Optimization for Hoeffding Tree”，主要用到一个 java 的流机器学习框架：MOA，可以在这里找到它的源码，我需要改变其中一个 learner “Hoeffding Tree”，让其获得更少的 Cache misses。为了能准确有效地评估结果，应该让其在对比测试中获得相同的输入 Stream，也就是将 RandomTreeGenerator 变成固定的一棵树。
